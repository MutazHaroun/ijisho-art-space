# Ijisho Art Space

## Description

Ijisho Art Space is a digital platform for showcasing Rwandan art and heritage.

## Technologies

* Frontend: React + Tailwind CSS
* Backend: Node.js + Express
* Database: PostgreSQL

## Installation

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm start

### Database

Run migration.sql using PostgreSQL

## Features

* Admin dashboard
* Artwork management
* Image upload
* Authentication (JWT)
* Gallery filtering
