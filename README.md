# Ijisho Art Space

A full-stack digital gallery platform for showcasing and managing Rwandan paintings and heritage artworks.

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express.js

Database
- PostgreSQL

## Features

- Admin dashboard
- Add / edit artworks
- Gallery display
- Contact form
- Image upload system
